# Proyecto_Investigacion_Proce_Imagenes_Medicas

Este trabajo propone la utilización de un modelo de segmentación tridimensional de tumores hepáticos mediante el uso de la arquitectura de nnU-Net en un dataset de 123 tomografías computarizadas para implementar un modelo de aprendizaje profundo autoconfigurable. La metodología se enfoca en un entrenamiento estándar y preprocesamiento, con adición de robustez mediante validación cruzada. El desempeño del modelo se evalúa a través de métricas de volumen, superficie y sensibilidad, complementando los resultados con una capa de Inteligencia Artificial Explicable (XAI) basada en mapas de calor de incertidumbre. Con esto, se busca no solo obtener una segmentación precisa de los tumores, sino también, proporcionar una herramienta interpretable visiblemente, que apoye la toma de decisiones clínicas y quirúrgicas.
El entrenamiento de la nnU-Net estaba planificado para realizarse con la configuración óptima recomendada por el autor de la arquitectura, sin embargo debido a la escasa potencia de Hardware disponible y tiempo de ejecucuión limitda del entorno alojado de Google Collab, el entrenamiento llegó solamente a su checkpoint de 50 épocas, entregando metricas que dejan bastante que desear, sin embargo, este hecho potencia la necesidad de implementar algoritmos de explicabilidad como el utlizado, que corresponde a mapas de calor de incertidumbre, para así, poder observar con que claridad el modelo asigna sus predicciónes, y la confiabilidad de estas.
Se utilizáron métricas cono sensibilidad, diferencia relativa de volumen y superficie normalizada de Dice, admás de incluir una función de pérdida mixta que se basa en la entropía cruzada y pérdida de Dice suavizada, el modelo además incorpora de manera nativa regularización L2, data Augmentation, Batch size = 2 y Early Stopping
